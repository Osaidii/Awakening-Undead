# Awakening-Undead
This is my Second Publoshed Game, A Zombie Shooter with a little mystery!  
Free to PLAY!  
You can download this game here or from my itch.io page [osaidii.itch.io](https://osaidii.itch.io)   
This game is a little horror and scary too!
  

Controls  
A / D – Move  
Shift – Run (Very Important)  
Space – Jump  
LMB - Shoot
R – Reload
E – Interact
Esc – Pause  


This is the project and source files of the game that is present in the repository Awakening-Undead in the Same Account!  
Free to Play, Free to USE!  
These files are meant to be used in Godot Engine!  
There is a plugin called Hackatime that is used by Hack Club! It might transfer hours to my key, please disable it in Project Setting and then Plugins!
