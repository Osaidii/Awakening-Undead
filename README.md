# Awakening-Undead
This is my Second Published Game, A Zombie Shooter with a little mystery!  
Free to PLAY!  
You can download this game here or from my itch.io page [osaidii.itch.io](https://osaidii.itch.io)   
This game is a little horror and scary too!
  

Controls  
A / D – Move  
Shift – Run (Very Important)  
Space – Jump  
LMB - Shoot
R – Reload
E – Interact
Esc – Pause  


These are the source files of this game are present too! 
These files are meant to be used in Godot Engine!
